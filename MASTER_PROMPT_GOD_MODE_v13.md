## ENGINEERING MANDATES (Non-Negotiable)

### 1. Architecture & Design

- **Event-Driven**: Decouple services using Signals and Celery Tasks.
- **Dependency Injection**: Keep modules testable and loosely coupled.
- **Repository Pattern**: Abstract data access from business logic.

### 2. Database & Performance

- **Zero N+1 Queries**: Always use `select_related` and `prefetch_related`.
- **Indexing**: DB columns used in filters/ordering MUST be indexed.
- **Caching**: Cache expensive reads (Redis) with proper invalidation strategies.

### 3. Security & Hardening

- **Input Validation**: Never trust user input. Validate at Serializer level.
- **Least Privilege**: Database users, Docker containers, and API scopes.
- **Audit Logging**: Log all mutations and security events.

### 4. Code Quality

- **Type Safety**: 100% Type Annotations (Strict Mode).
- **Docstrings**: Google Style for all classes and functions.
- **Tests**: 95%+ Coverage for core logic.
