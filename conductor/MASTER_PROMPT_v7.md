## 🏗️ Architecture Standards

- **Service Layer**: Business logic lives in `services.py`, NOT views.
- **Typed Python**: strict `mypy` compliance.
- **Query Optimization**: `select_related`, `prefetch_related` enforced.
- **Security**: JWT, RBAC, input validation.

### Frontend (Flutter)

- **Riverpod 2.0**: Immutable state management.
- **Clean Architecture**: features/ -> data, domain, presentation.
- **Glassmorphism**: Premium UI aesthetic.
- **Performance**: Const constructors, minimal rebuilds.
