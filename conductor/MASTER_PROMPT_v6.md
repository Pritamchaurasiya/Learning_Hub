| -------------- | -------------------------------------- | ---------------------------------------------------- |

### Research Scientist Security Mindset

1. **Assume Breach**: Design systems assuming attackers are inside.
2. **Defense in Depth**: Multiple security layers.
3. **Least Privilege**: Minimal permissions by default.
4. **Audit Everything**: Log security-relevant events.
5. **Fail Secure**: On error, deny access (not grant).
