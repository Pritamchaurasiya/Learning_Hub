## 2. Web Compatibility (The `dart:io` Issue)
**Concept**: Dart code running on the web cannot access the underlying OS file system directly. `dart:io` is not supported on the web.
**Solution**: use `universal_io` or conditional imports (`if (dart.library.io) ...`).

## 3. SEO & Performance for Web
**Concept**: Single Page Apps (SPAs) often suffer in SEO.
**Optimization**:
- Use proper `<title>` and `<meta>` tags in `index.html`.
- Use specific semantic HTML tags where possible.
- Optimize asset loading.

## 4. Security Practices
- **Input Validation**: Never trust user input.
- **Dependency Scanning**: Use tools like `snyk` to find vulnerable packages.
- **Obfuscation**: Hide code logic in release builds.
