# 🎓 Anti-Gravity Learning Curriculum: From Zero to Systems Architect

This file contains the "Tree of Knowledge" for this project. As we build, we document the *why* and *how* here.

## 🏗️ Module 1: Systems Architecture & Design
* **Concept**: Domain-Driven Design (DDD) & Clean Architecture.
* **Why**: To prevent "Spaghetti Code" in large apps. Separates Logic, Data, and UI.
* **Implementation in Project**:
    - `conductor/apps/`: Bounded Contexts.
    - `conductor/core/use_cases/`: Business Logic independent of Framework.
    - `flutter/lib/src/features/`: Feature-first architecture.

## 🐍 Module 2: Advanced Python & Django Internals
* **Concept**: The MRO (Method Resolution Order) and Metaclasses.
* **Why**: Critical for understanding DRF Serializers and Django Models.
* **Security Spotlight**:
    - **SQL Injection**: How existing ORMs prevent it vs raw SQL.
    - **Serialization Attacks**: Why we use `yaml.safe_load`.
* **Performance**:
    - `select_related` (JOIN) vs `prefetch_related` (Python-level merge).
    - Connection Pooling in PostgreSQL.

## 🐦 Module 3: Flutter Internals & Rendering
* **Concept**: The Element Tree vs Render Objects.
* **Why**: Understanding how Flutter paints pixels (Skia/Impeller) lets us optimize 60fps.
* **Optimization**:
    - `const` Widgets: Pre-compile canonical instances.
    - `RepaintBoundary`: Isolate costly paints (e.g., video flows).

## 🛡️ Module 4: Cyber-Security & Forensics
* **Concept**: The CIA Triad (Confidentiality, Integrity, Availability).
* **Defensive Engineering**:
    - **OWASP Top 10**: Preventing Injection, Broken Auth, XSS.
    - **JWT Anatomy**: Header.Payload.Signature. Why RS256 > HS256.
    - **Rate Limiting**: Token Bucket Algorithm in Redis.

## 🤖 Module 5: AI & Machine Learning Engineering
* **Concept**: RAG (Retrieval-Augmented Generation).
* **Implementation**:
    - PgVector in PostgreSQL for embedding storage.
    - Cosine Similarity searches.
    - LangChain for orchestrating context.
