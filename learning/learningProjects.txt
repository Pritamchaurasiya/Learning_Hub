# OpenCode CLI & Terminal Agentic AI Masterclass

Welcome to the deep-dive learning guide for OpenCode AI. This tool represents the next generation of open-source coding agents.

---

## 1. What is OpenCode?
OpenCode is an open-source, terminal-based AI coding agent. It is designed to:
- **Operate Locally**: It runs in your terminal, having direct access to your local files and shell.
- **TUI (Terminal User Interface)**: It provides a rich interactive environment within the terminal (built with libraries like Charm Bracelet's Bubble Tea).
- **Multi-Model Support**: You can connect it to Gemini, Claude, or OpenAI.

### Why use OpenCode?
While tools like Claude Code are official and powerful, OpenCode offers an open-source alternative that is highly customizable and supports a wider range of models out of the box.

---

## 2. Key Commands & Workflow
- `opencode`: Launch the main interface.
- `/connect`: Connect your AI provider (Gemini, Anthropic, etc.).
- `/init`: Initialize a project to help the AI understand your specific architecture.
- `/run <prompt>`: Execute a task without entering the full TUI.

---

## 3. Advanced Concepts

### Level 1: TUI Fundamentals
- **True Color & Unicode**: OpenCode relies on modern terminal features. If symbols look broken, ensure you're using a font like "Nerd Fonts".
- **Shell Integration**: How the AI translates your request into a series of `cd`, `ls`, and `git` commands.

### Level 2: Project Indexing
- **agents.md**: When you run `/init`, OpenCode creates this file. It's like a "manual" for the AI to understand your project's soul (tech stack, folder structure, main goals).

### Level 3: Security & Trust
- **Sandboxing**: Unlike cloud-based AIs, OpenCode runs on *your* machine.
- **Safety Prompts**: Always verify what code the agent wants to write or delete.

---

## 4. Troubleshooting
1. **"Command not found"**: Ensure `C:\Users\shiva\AppData\Roaming\npm` is in your system PATH.
2. **Key Errors**: If the AI stops responding, check your API usage limits or refresh the key with `/connect`.
