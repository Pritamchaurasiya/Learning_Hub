# /m - ML/AI & Automation Enhancement Workflow

This workflow performs deep end-to-end analysis focusing on ML, AI, and automation systems.

## Steps

// turbo-all

1. **Deep Project Analysis**

   - Analyze datasets, preprocessing logic, models, training pipelines
   - Review hyperparameters, configs, codebase, tests, infrastructure
   - Identify bugs, data issues, leakage, bias, broken workflows

2. **Security & Performance Audit**

   - Scan for security risks and performance bottlenecks
   - Run static analysis: `flutter analyze`
   - Check for vulnerabilities in dependencies

3. **Enhancement & Fix Implementation**

   - Fix all identified bugs and issues
   - Implement best-practice techniques
   - Feature engineering, validation strategies, experiment tracking

4. **Code Quality Refactoring**

   - Refactor into clean, modular, decoupled components
   - Add comprehensive test coverage
   - Ensure production-grade stability

5. **Future-Ready Improvements**

   - Research and propose next-step tools and techniques
   - Integrate AI/ML enhancements
   - Strengthen robustness, scalability, observability

6. **Verification & Testing**
   - Run all tests: `flutter test`
   - Verify all features are working
   - Ensure no critical issues remain
